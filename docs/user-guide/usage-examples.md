# 📚 使用示例集合

## 基础使用

### 1. 交互模式（默认）

```bash
uv run timeflow add
```

### 2. 快速模式

```bash
uv run timeflow add --no-edit
```

### 3. 直接输入

```bash
uv run timeflow add --text "完成项目部署"
```

## 高级使用

### 按分类查询

```bash
uv run timeflow list --category 学习 --limit 20
```

### 按日期范围查询

```bash
uv run timeflow list --range 2025-11-01 2025-11-30
```

### 查看本周统计

```bash
uv run timeflow stats --week
```

### 启动 Web 界面

```bash
uv run timeflow web
```

然后访问：http://127.0.0.1:8000
